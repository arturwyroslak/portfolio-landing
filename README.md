# Portfolio Landing

Nowoczesny, animowany landing page dla portfolio developera. Statyczna strona gotowa do hostingu na GitHub Pages.

Instrukcja:
1. Skopiuj repozytorium.
2. Dostosuj treści w index.html (Imię, projekty, linki).
3. Wyślij na GitHub Pages: ustaw branch `main` jako stronę.

Technologie: HTML, CSS, Vanilla JS, Google Fonts.
